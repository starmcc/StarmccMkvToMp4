## 介绍

![logo](./logo.png)

快速无损将Mkv格式视频转Mp4格式

保留原有音轨，字幕单独输出

![](./desc.png)

## 下载

[**Releases-点击进入下载页面**](https://github.com/starmcc/StarmccMkvToMp4/releases)

请下载带有【**windows.zip**】的压缩包，解压后即可食用。

如果您要更新，只需要替换【`StarmccMkvToMp4.exe`】即可。
